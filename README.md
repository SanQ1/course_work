Предметна область: онлайн магазин.

Учасники: Дячук Олександр Романович

Мова програмування:     Javascript
ORM:                    Prisma
Фреймворк тестування:   Jest

Інструкції з налаштування:
    - Клонуйте репозиторій
    - Запустіть команду: `docker-compose up --build`

Проєкт складається з каталогів: docs, src, test.
docs містить усю документацію.
src містить весь код для роботи з базою даних.
test містить всі тести.
